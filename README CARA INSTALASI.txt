--- CARA INSTALL APLIKASI SI-AP ---

1. Ekstrak rar apotek_sf.rar ke Folder C:\xampp1\htdocs (disesuaikan nama xampp di komputer user)
2. Import file apotek_sf.sql ke dalam localhost/phpmyadmin
3. Start config XAMPP (Apache dan MySQL)
4. Buka browser (Chrome / Mozilla)
5. Ketik localhost/apotek_sf
6. Login ke dalam aplikasi
7. User dan Password default : admin | admin
